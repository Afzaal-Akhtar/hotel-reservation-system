# Hotel Reservation System (HRS)

## Description

Hotel Reservation System (HRS) is a Java-based desktop application developed using JavaFX. The system helps manage hotel operations such as customer registration, room booking, reservation management, and record keeping. This project was developed as a semester project to demonstrate Object-Oriented Programming (OOP) concepts and file handling in Java.

## Features

* Add and manage customer information
* View available rooms
* Book hotel rooms
* Manage reservations
* Store and retrieve data using file handling
* User-friendly JavaFX graphical interface

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Open the project in NetBeans.

3. Configure JavaFX libraries.

4. Run the Main class.
